# Proxyreaper
A proxy scrapping tool

### Dependencies

- beautifulsoup4
- ipaddress
- selenium

### How to install 

```
pip -r requirements.txt
```

```
python proxyreaper.py
```

### You can it in scripts

```
from proxyreaper import ProxyReaper

obj = ProxyReaper(10)
action = obj.get_proxies()
proxylist = action.get_proxy_list()
```
